# Промты для генерации картинок (Google Gemini)

Как пользоваться: генерируешь картинку по промту → сохраняешь под указанным именем → кладёшь в указанную папку → коммит-пуш → Railway пересоберёт, картинка подхватится автоматически (у всех слотов есть фолбэк-заглушка, ничего не сломается, пока файла нет).

## Общий стиль (добавляй в конец каждого промта)

> Фотореалистично, тёплый солнечный свет «золотого часа», казахстанская степь и сельское хозяйство, цветовые акценты — глубокий зелёный (#07663D) и золото (#C9A21C), современно и оптимистично. Без текста, без логотипов, без водяных знаков. Люди — центральноазиатской внешности, в рабочей одежде, уверенные и доброжелательные. Горизонтальная композиция 16:9, высокое разрешение.

Для широких шапок страниц проси 21:9 («ультраширокая панорама»).

## Видео карточек программ — `web/public/img/programs/{id}.mp4` (image-to-video, hover-луп)

Карточка при наведении проигрывает `/img/programs/{id}.mp4` (зациклен, без звука), при уходе — сброс на фото. Файла нет → остаётся статичное фото (ничего не ломается). Делается через **Gemini image-to-video**: на вход — готовый стилл `{id}.jpg`, на выходе — лёгкая анимация сцены.

**Технически (как у существующих):** 1280×720, H.264 (mp4), **без звука**, **6 сек**, бесшовный луп (первый ≈ последний кадр), ≤2.5 МБ.

**Единый суффикс движения (добавляй в конец):**
> Плавное, едва заметное движение, бесшовный зацикленный ролик ~6 секунд, камера почти статична (очень лёгкий наезд или панорама), без резких склеек и без смены плана, без появления текста и логотипов, тёплый свет золотого часа, кинематографично. 1280×720, без звука.

## Технические требования

- Формат: JPG, ширина ≥1600px (шапки ≥2000px). **Карточки целей — PNG с прозрачным фоном**, ~600×600px достаточно.
- Вес: желательно ≤400КБ (Gemini отдаёт большие — прогони через squoosh.app или `sips -Z 2000`).
- Если картинка не подходит по тону (слишком «стоковая», мрачная, нерезкая) — лучше перегенерить, чем ставить.
