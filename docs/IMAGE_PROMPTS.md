# Промты для генерации картинок (Google Gemini)

Как пользоваться: генерируешь картинку по промту → сохраняешь под указанным именем → кладёшь в указанную папку → коммит-пуш → Railway пересоберёт, картинка подхватится автоматически (у всех слотов есть фолбэк-заглушка, ничего не сломается, пока файла нет).

### Ночная версия — `hero-main-night.jpg` (подмена в тёмной теме)

Та же сцена ночью: при переключении сайта в тёмную тему дневной кадр плавно сменяется ночным — комбайн «убирает в ночи» с включёнными фарами. **Композиция и кадрирование должны совпадать с дневной версией** (комбайн справа, тот же широкий формат), иначе подмена будет «прыгать». Для этого кадра **общий «золотой час» суффикс НЕ добавляй** — здесь ночь.

> Фотореалистичный широкий кадр, глубокая ночь: современный зерноуборочный комбайн ведёт уборку пшеницы с ВКЛЮЧЁННЫМИ ФАРАМИ — мощные лучи света прорезают темноту и поднятую пыль над полем, тёплые золотистые блики на колосьях и на корпусе машины. ## Технические требования

- Формат: JPG, ширина ≥1600px (шапки ≥2000px). **Карточки целей — PNG с прозрачным фоном**, ~600×600px достаточно.
- Вес: желательно ≤400КБ (Gemini отдаёт большие — прогони через squoosh.app или `sips -Z 2000`).
- Если картинка не подходит по тону (слишком «стоковая», мрачная, нерезкая) — лучше перегенерить, чем ставить.
- Имена файлов менять нельзя — на них завязаны компоненты.
