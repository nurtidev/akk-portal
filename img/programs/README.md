# Фото программ для карточек

Сюда кладутся баннеры программ. Карточка ищет файл по `id` программы:

| Файл | Программа | Образ |
|---|---|---|
| `ken_dala.jpg` | Кең дала | пшеничное поле / посевная |
| `ken_dala_2.jpg` | Кең дала 2 | пшеничное поле с техникой, уборка |
| `agrobusiness.jpg` | Агробизнес | агро-инфраструктура, элеватор, теплицы |
| `agrobusiness_2.jpg` | Агробизнес 2.0 | современный агрокомплекс / инвестпроект |
| `igilik_bereke.jpg` | Игілік и Береке | племенной КРС на пастбище |
| `feedlot_poultry.jpg` | Откормплощадки и птицефабрики | откормплощадка / птицефабрика |
| `isker.jpg` | Іскер | начинающий фермер / трактор в поле |

## Требования к изображениям

- **Формат:** `.jpg`, имя строго = `<id>.jpg` (см. таблицу).
- **Размер:** 1200×750 px (соотношение 16:10), вес < 200 КБ (сжать).
- **Стиль (единый для всех 7):** фотореализм, дневной мягкий свет, степь/АПК Казахстана,
  тёплая цветокоррекция. Без текста, без логотипов, без крупных лиц людей.
- Карточка сама накладывает зелёный градиент снизу — отдельный оверлей не нужен.

## Fallback

Если файла нет — карточка покажет иконку категории на зелёном градиенте
(`onerror` убирает `<img>`). То есть верстка не ломается без фото.

> Локальная разработка (`serve -s .`) и прод (nginx) отдают эту папку —
> в Dockerfile добавлено `COPY img/ ...`.
