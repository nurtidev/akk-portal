# Картинки для ленты «АКК в цифрах» (ImpactMarquee)

Компонент ищет `/img/impact/<key>.png` для каждого факта. Пока файла нет —
в карточке показывается пунктирный плейсхолдер с именем файла. Положи готовые
PNG сюда — подхватятся автоматически.

**Стиль и полные промты** — в канон-доке: [`docs/IMAGE_PROMPTS.md`](../../../../docs/IMAGE_PROMPTS.md)
→ раздел «Лента «АКК в цифрах»». Стиль — clay / soft-3D на прозрачном фоне
(та же семья, что «Карточки целей» и «Масштаб АКК»).

Файлы: `meat.png`, `oil.png`, `greenhouses.png`, `borrowers.png`, `bread.png`, `budget.png`.
Формат: PNG с прозрачным фоном, ~600×600, ≤400 КБ.
